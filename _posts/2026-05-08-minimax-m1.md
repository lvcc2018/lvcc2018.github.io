## 核心创新

### 1. Hybrid Attention

```
Lightning Attention (O(n))  → 处理长程依赖、百万级上下文
Softmax Attention (O(n²))   → 处理精确推理、局部模式

交替使用，各取所长
```

这与 DeepSeek 的路线不同——DeepSeek 用 MLA 压缩 KV + DSA 稀疏化，MiniMax 用 Linear Attention 从数学上消除 O(n²)。

### 2. CISPO 算法

CISPO (Clipped Importance Sampling Policy Optimization) 裁剪重要性采样权重，训练效率远超 PPO/GRPO：

> 512 H800 GPU，3 周，$534,700 → 完成 456B MoE 的 RL 训练

### 3. 对比

|  | MiniMax-M1 | DeepSeek-R1 |
|--|-----------|-------------|
| 参数 | 456B/45.9B | 671B/37B |
| 上下文 | **1M** | 128K |
| 注意力 | Hybrid (Linear+Softmax) | MLA (Softmax) |
| RL 算法 | CISPO | GRPO |
| 训练成本 | $534,700 | — |

## 启示

MiniMax 从创业公司角度证明了：**不一定要走 DeepSeek 的 MLA 路线，Linear Attention 在大规模上同样可行。** 两种路线并存说明了长上下文不是一个已解决的问题——架构探索仍在早期。
