## 概述

2026年4月24日，DeepSeek 发布了 V4 系列模型（V4-Pro + V4-Flash），核心突破：

1. **1M token 上下文窗口**（V3 为 128K，提升 8 倍）
2. **Engram 架构 + mHC 框架**：全新的长上下文处理范式
3. **参数量 ~1T**（V3 的 1.6 倍）

## Engram 架构

"Engram" 一词源于神经科学——记忆痕迹（memory trace）。核心思想是将上下文切分为可寻址的"记忆单元"，在架构层面实现类似操作系统的虚拟内存管理。

### 与 MLA/DSA 的关系

```
MLA  → 将 KV 压缩为低秩潜向量（存储面）
DSA  → 可训练稀疏注意力选择需要唤起的记忆（检索面）
Engram → MLA + DSA 的统一架构框架
mHC   → 多尺度层次化上下文管理（token/段落/文档）
```

## 成本优势

V4 延续了 DeepSeek 的低成本传统——推理成本压到海外模型的 5-20%。这是从 V2（训练成本 -42.5%）→ V3（$5.6M）→ V4 一贯的工程哲学。

## 思考

V4 的技术路线说明了一个趋势：**长上下文不是靠堆显存，而是靠架构创新**。MLA 压缩存储、DSA 稀疏检索、Engram 记忆管理——每一步都在降低 O(n²) 的实际开销。
